主動式抗噪

MID A.N.C 具有主動式的抗噪功能，透過內建的四個麥克風來偵測並辨別環境噪音，屏蔽你不想聽到的聲音

支援藍牙 aptX

透過藍牙aptX技術來無線連接MID A.N.C.主動式抗噪耳機。使用藍牙aptX技術，不僅能用高位元素播放，同時還能降低影音不同步的問題，讓觀影體驗更順暢，同時還提供約10公尺自由移動的無線連線距離

超越20小時續航力

同時啟用藍牙和抗噪功能，MID A.N.C 主動式抗噪耳機能持續播放音樂達20小時，若將抗噪功能關閉，電池續航力可延長至30小時以上。主動式抗噪功能關閉並不會影響到藍牙aptX連接

客製單體

客製40mm動態驅動單體帶出結實有力的音響，搭配上充足而清晰低音的完美平衡，滿足那些追求優良音質的渴望

五向控制鈕

耳機開關、播放暫停、快轉或是調整音量，都可以透過五向控制鈕，通話功能也一併在內，接聽、拒絕或是掛斷電話，都在彈指之間

內建通話功能

內建兩組被動式降噪麥克風，讓你能透過 MID A.N.C 與他人對話或是錄製語音備忘錄

與好友共享音樂

當使用藍牙無線播放音樂時，透過額外的3.5mm音源孔，就能與其他人分享音樂

旅行收納盒

靈感取自吉他琴盒，MID A.N.C 擁有專屬的旅行收納盒，外層為耐磨的人工皮革，內裏採用柔軟的紅色天鵝絨，給 MID A.N.C 最好的收納空間，當不使用時，收納盒也可以折疊收納，輕巧不占空間

折疊設計

當不使用耳機時，只要將耳機折疊收納至附贈的專屬收納盒中，只佔隨身行李的極小空間，是每日通勤或是長程旅行的完美伴侶

人體工學設計

MID A.N.C 經由陽極化處理後的3D立體金屬樞紐，可以依照頭型自由的旋轉調整至最符合人體工學
